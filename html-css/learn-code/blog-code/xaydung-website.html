những thành phần trong trang website 1 header (đầu trang thanh menu) 2 navigation (cái phần ghi ở trên menu) 3 breadcrumb là phần hướng chỉ đang ở đâu (vd lớp học > khoá học > html-css) 4 sidebar là navigation là menu danh sách nằm ở bên trái hoặc bên phải 5 content là nội dung chính 6 slider là thanh trượt 8 banner là ảnh động để quảng cáo 7 footer là chân trang
